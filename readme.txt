# Trivia Wild Animals
El target de esta trivia son niños, para que jueguen y aprendan algunos datos de interés sobre los animales. A continuación un detalle de lo realizado:

##Lunes:
Diseño de prototipo en Miro, creación de la encuesta, mediante form y casilla "tu nombre".
Este día llegamos hasta el Hito 1, generando una trivia de una sola screen, la cual indicaba las repuestas correctas mediante alert, y también un alert en caso de respuesta incorrecta, indicando cual era la respuesta acertada.
